2、SpringBoot入门程序

SpringBoot 2.5.5 官方入门文档

一、系统要求

  • Java 8 & 兼容 java17
  • Maven 3.5+

二、maven 设置

在 maven 安装目录中的 conf/setting.xml 文件中设置下载镜像和编译jdk版本

三、HelloWorld 项目

需求：浏览发送 /hello 请求，响应 Hello，Spring Boot 2

2.1 创建 maven 工程

使用 idea 创建一个普通 maven 工程，不使用任何模板

2.2 修改 pom.xml 文件

使用官方文档中提供的 pom.xml 文件，引入 springboot 父依赖

2.3 引入依赖

引入 web 开发所需要的起步依赖：spring-boot-starter-web

2.4 创建主程序

2.5 编写业务

2.6 测试

直接运行 main 方法

2.7 简化配置

通过配置文见 application.properties 可以配置一些数据，如 tomcat 运行的端口

2.8 简化部署

springboot 由于内部有嵌入式的 tomcat 服务器，因此打包时可以直接打成 jar 包，在目标服务器直接使用 java -jar 命令执行就行

但是首先要先在 pom 文件中加入打包的插件

之后使用 maven 的 package 命令把项目打成 jar 包，直接在目标服务器执行即可。
