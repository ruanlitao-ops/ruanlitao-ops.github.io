hexo部署到coding

之前我们已经把hexo托管在github了，但是github是国外的，而且百度的爬虫是不能够爬取github的，所以如果你希望你做的博客能够在百度引擎上被收录，而且想要更快的访问，那么可以在国内的coding page做一个托管，这样在国内访问就是coding page，国外就走github page。

1、申请一个 coding 账户

coding 官方网址

2、登录工作区

3、新建一个项目

创建完成

4、创建一个仓库

创建完成

5、为当前仓库添加 key ssh

此时就可以在 cmd 中查看到 key shh 生成的位置了

之后就可以将 id_rsa.pub 文件中的内容添加到 coding 中

添加后，检查一下是不是添加成功

6、修改 _config.yml 文件

添加 coding 中的 git 地址

7、将 hexo 部署到 coding

此时还会有弹框出现要你登录 coding

8、开启 coding 的网站托管

若没有该选项，就在左下角项目设置中开启该功能

9、新建网站

10、添加自定义域名

在阿里云或者腾讯云申请一个域名，同时申请 SSL 证书，添加域名解析，到当前网站地址。

到这里，通过自己的域名就可以访问到博客网站了。
